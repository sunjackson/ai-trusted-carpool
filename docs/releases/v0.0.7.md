> [!WARNING]
> 这是未签名正式 Release，采用手动下载与安装，不会进入应用内自动更新。“正式版”仅表示稳定版本通道，不表示已经通过 Windows Authenticode、Apple Developer ID 或公证。Windows SmartScreen 和 macOS Gatekeeper 可能显示未知发布者警告；安装前请使用本 Release 的 `SHA256SUMS.txt` 核对文件。

## 中文

- 修复 Windows 双击桌面图标后没有可见窗口的问题。应用现在立即显示内置深色启动页，不再等 WebView2 页面完成加载后才首次显示；页面完成加载后仍会恢复并聚焦主窗口。
- 修复部分 Windows 网络发车时报“无法连接协调服务”的问题。系统 DNS 和代理仍是主路径；只有精确的 `https://p2p.cnaigc.ai:443` 在连接建立失败时才使用官方 IP 兜底，URL 域名保持不变，因此 TLS SNI 和证书校验仍针对 `p2p.cnaigc.ai`。
- 回退不会用于自建协调服务、相似域名、非 443 端口、HTTP 错误响应或普通超时，避免把自建流量导向官方服务，并避免错误重放已经送达的发车或信令 POST。
- 使用回退时只写入固定脱敏诊断，不记录完整 URL、上车码、请求正文、SDP 或 ICE 候选。

安装和使用前请阅读仓库中的 `LEGAL.md`。本版本没有 Authenticode、Developer ID 或 Tauri 自动更新签名；请只从本项目 GitHub Releases 下载并校验 SHA-256。

## English

> This is an unsigned stable GitHub Release distributed through manual download and installation. “Stable” describes the version channel only; it does not claim Windows Authenticode, Apple Developer ID signing, or notarization. The build is excluded from in-app updates. Windows SmartScreen and macOS Gatekeeper may show unknown-publisher warnings; verify the download against this Release's `SHA256SUMS.txt` before installing.

- Fixes a Windows launch that could leave no visible window after double-clicking the desktop shortcut. The native window now presents the built-in dark boot surface immediately instead of waiting for the WebView2 page to finish loading; completion still restores and focuses the main window.
- Fixes “Unable to connect to coordinator” while hosting on some Windows networks. System DNS and proxy configuration remain the primary path. Only the exact `https://p2p.cnaigc.ai:443` endpoint can use the official IP fallback after a connection-establishment failure, while the URL hostname remains unchanged so TLS SNI and certificate verification still target `p2p.cnaigc.ai`.
- The fallback is never enabled for self-hosted coordinators, lookalike domains, non-443 ports, HTTP error responses, or ordinary timeouts. This prevents self-hosted traffic from being redirected and avoids replaying hosting or signaling POSTs that may already have arrived.
- A fallback event writes only a fixed redacted diagnostic message. It never records the full URL, seat code, request body, SDP, or ICE candidates.

Read `LEGAL.md` before installing or using the application. This build has no Authenticode, Developer ID, or Tauri updater signature; download it only from this project's GitHub Releases and verify its SHA-256 digest.

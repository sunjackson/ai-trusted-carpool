## 中文

- 修复 Windows 首次打开长期白屏：原生窗口先保持隐藏并使用深色背景，主页面加载完成后才显示；即使前端初始化较慢，也会先显示深色静态启动页，而不是空白 WebView。
- 修复部分 Windows 网络中乘客一直停在“正在上车”的问题：官方 TURN 域名仍为主地址；当 Clash/TUN 等 DNS 返回 WebRTC 无法使用的假地址时，仅为精确的 `p2p.cnaigc.ai` 增加官方 IP 兜底。Offer/Answer 会短暂等待 ICE 候选收集，同时继续兼容增量候选信令。
- 单个重复或当前 WebView2 不兼容的 ICE 候选不再中断后续候选处理；信令错误只写入脱敏诊断，不记录 SDP 或候选内容。
- “我的拼车记录”现可点击打开详情，查看发车/上车状态、发车模式、起止时间、工具、座位和昵称；仍可恢复或重新上车的记录保留原操作入口。
- WebRTC 真实网络验收覆盖直连、TURN UDP 与 TURN TCP；测试脚本完成浏览器和临时文件清理后会确定性退出。

如果已经安装 `v0.0.6-test.1`，请手动下载本版本覆盖安装。安装和使用前请阅读仓库中的 `LEGAL.md`。本版本没有 Authenticode、Developer ID 或 Tauri 自动更新签名；请只从本项目 GitHub Releases 下载并校验 SHA-256。

## English

> This is an unsigned stable GitHub Release distributed through manual download and installation. “Stable” describes the version channel only; it does not claim Windows Authenticode, Apple Developer ID signing, or notarization. The build is excluded from in-app updates. Windows SmartScreen and macOS Gatekeeper may show unknown-publisher warnings; verify the download against this Release's `SHA256SUMS.txt` before installing.

- Fixes the persistent blank window on the first Windows launch. The native window stays hidden with a dark background until the main page finishes loading; a dark static boot surface remains available while frontend initialization is slow.
- Fixes a Windows passenger flow that could remain on “Joining.” The official TURN hostname remains primary; when Clash/TUN-style DNS returns a fake address unusable by WebRTC, only the exact `p2p.cnaigc.ai` host receives the official IP fallback. Offers and answers briefly wait for ICE gathering while retaining trickle-ICE compatibility.
- One duplicate or WebView2-incompatible ICE candidate no longer aborts later candidates. Signaling failures enter redacted diagnostics without recording SDP or candidate contents.
- Ride-history cards now open a detail dialog with host/passenger state, hosting mode, time window, tools, seat, and nickname. Existing restore and rejoin actions remain available when valid.
- Real-network WebRTC acceptance covers direct, TURN UDP, and TURN TCP paths. The harness now exits deterministically after cleaning up its browser and temporary files.

If `v0.0.6-test.1` is already installed, manually download this build and install it over the previous test build. Read `LEGAL.md` before installing or using the application. This build has no Authenticode, Developer ID, or Tauri updater signature; download it only from this project's GitHub Releases and verify its SHA-256 digest.

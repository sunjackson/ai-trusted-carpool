# 可信拼车 v0.0.8 / Trusted Carpool v0.0.8

> [!WARNING]
> 这是未签名正式 Release，采用手动下载与安装，不会进入应用内自动更新。“正式版”仅表示版本通道稳定，不表示已经通过 Windows Authenticode、Apple Developer ID 或公证。Windows SmartScreen 和 macOS Gatekeeper 可能显示未知发布者警告；安装前请使用本 Release 的 `SHA256SUMS.txt` 核对文件。

## 中文

- 修复 Windows 首次启动持续黑窗或没有可用界面的问题。原生主窗口保持隐藏，只有 React 首次渲染完成后才显示；前端未返回就绪信号时回退到内置静态启动页。
- 双击第二实例、托盘恢复和上车深链不会越过启动门控，不会在 WebView2 尚未绘制时强制显示窗口。
- 拼车客户端残留恢复不再阻塞 Tauri 主线程。Windows 进程枚举只执行一次并限制为 8 秒，窗口定位及 `taskkill` 限制为 5 秒；超时或无法验证 PID、启动时间、可执行文件与 profile 时保留恢复记录，不会猜测或关闭普通客户端。
- 启动回滚或实例注册表异常时采取失败关闭：无法确认客户端已经退出时，不恢复或删除它可能仍在使用的配置；损坏注册表会先隔离再只执行不终止进程的旧备份恢复，未知版本则原样保留并停止自动恢复。
- Windows 安装器显式使用 Tauri 官方 WebView2 bootstrapper 检查。CI 会在干净应用数据下连续验证首次、第二次和快速双击第二实例均完成 React 就绪握手、呈现可访问窗口并保持单实例。
- 上车链接会先尝试唤起本机客户端；浏览器未观察到客户端接管时，自动识别 Windows、macOS、Linux 或移动端，并提供固定到本项目 `v0.0.8` Release 的安装包与 SHA-256 清单。

安装和使用前请阅读仓库中的 `LEGAL.md`。本版本没有 Authenticode、Developer ID 或 Tauri 自动更新签名；请只从本项目 GitHub Releases 下载并校验 SHA-256。

## English

> This is an unsigned stable GitHub Release distributed through manual download and installation. “Stable” describes the version channel only; it does not claim Windows Authenticode, Apple Developer ID signing, or notarization. The build is excluded from in-app updates. Windows SmartScreen and macOS Gatekeeper may show unknown-publisher warnings; verify the download against this Release's `SHA256SUMS.txt` before installing.

- Fixes a Windows first launch that could remain on a black window or never present a usable interface. The native main window stays hidden until React commits its first render, with the packaged static boot surface as a fallback when frontend readiness is not received.
- Second-instance activation, tray restore, and join deep links cannot bypass the startup gate or expose the window before WebView2 has painted.
- Stale managed-client recovery no longer blocks Tauri's UI thread. Windows process discovery runs once with an eight-second limit; window focus and `taskkill` have five-second limits. If PID, start time, executable, and profile cannot be verified, the recovery record is retained and ordinary clients are not guessed or closed.
- Launch rollback and abnormal instance registries now fail closed. Configuration remains untouched whenever a client may still be using its carpool profile; malformed registries are quarantined before process-free legacy backup recovery, while unknown versions remain untouched and stop automatic recovery.
- The Windows installer explicitly retains Tauri's official WebView2 bootstrapper check. CI now covers clean first and second launches plus a rapid second-instance handoff, requiring the React-ready handshake, an accessible window, and a single surviving application instance.
- A join link first attempts to open the installed client. If the browser does not observe a handoff, it detects Windows, macOS, Linux, or mobile and offers installers and checksums pinned to this project's `v0.0.8` Release.

Read `LEGAL.md` before installing or using the application. This build has no Authenticode, Developer ID, or Tauri updater signature; download it only from this project's GitHub Releases and verify its SHA-256 digest.

> [!WARNING]
> 这是未签名测试预发布版，仅供主动下载、测试和自行部署。它不会进入应用内自动更新。Windows SmartScreen 和 macOS Gatekeeper 可能显示未知发布者或未公证警告；安装前请使用本 Release 的 `SHA256SUMS.txt` 核对文件。

## 中文

- 新增客户端实例级管理、持久化账号路由健康状态和脱敏诊断导出。
- 新增账号导入预览、加密备份与原子恢复。

## English

> This is an unsigned test prerelease for opt-in downloads, testing, and self-hosting only. It is excluded from in-app updates. Windows SmartScreen and macOS Gatekeeper may show unknown-publisher or notarization warnings; verify the download against this Release's `SHA256SUMS.txt` before installing.

- Adds per-client instance management, persistent account route health, and redacted diagnostic exports.
- Adds account import previews, encrypted backups, and atomic restore.
- Signed Windows/Linux updater code and the active-ride installation gate are implemented, but this test Release has no production certificate or updater signing secret, so it contains no `.sig` files or `latest.json`.
- The macOS universal DMG, Windows x64 NSIS, and Linux x64 DEB/AppImage are unsigned manual downloads. The automatic updater never bypasses signature failures to install them.

Read `LEGAL.md` before installing or using the application. This release adds no telemetry and never sends account credentials, session secrets, prompts, or response bodies to the update service.

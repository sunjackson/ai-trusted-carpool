# 贡献者公约（Contributor Covenant）行为准则

## 我们的承诺

作为成员、贡献者和维护者，我们承诺让每个人参与社区的体验都不受骚扰，无论其年龄、体型、可见或不可见的残障、民族、性征、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、种姓、肤色、宗教信仰、性取向如何。

我们承诺以有助于建设开放、友善、多元、包容、健康社区的方式行事和互动。

## 我们的准则

有助于营造积极环境的行为示例包括：

- 对他人表现出同理心和善意
- 尊重不同的意见、观点和经验
- 提出并大方接受建设性反馈
- 为自己的错误承担责任、向受影响的人道歉，并从经验中学习
- 关注对整个社区最有利的事情，而不仅仅是个人

不可接受的行为示例包括：

- 使用与性有关的言语或图像，以及任何形式的性骚扰
- 挑衅、侮辱或贬损的评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（如物理地址或电子邮件地址）
- 在专业环境中可以被合理认定为不当的其他行为

## 执行责任

社区维护者有责任澄清并执行可接受行为的标准，并将对任何他们认为不当、威胁、冒犯或有害的行为采取适当且公平的纠正措施。

社区维护者有权利和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、维基编辑、议题和其他贡献，并在适当时说明理由。

## 适用范围

本行为准则适用于所有社区空间，也适用于个人在公共空间代表社区的情形（例如使用官方电子邮件地址、通过官方社交媒体账号发帖，或作为指定代表参加线上线下活动）。

## 执行

辱骂、骚扰或其他不可接受的行为可通过 GitHub 仓库的 [私密安全报告渠道](https://github.com/sunjackson/ai-trusted-carpool/security/advisories/new) 或向维护者 @sunjackson 报告。所有投诉都会得到及时、公正的审查和调查。

所有社区维护者都有义务尊重事件报告者的隐私与安全。

## 处理指引

社区维护者将遵循以下社区影响处理指引，确定违反本行为准则的行为的后果：

1. **纠正**：使用不当语言或其他不professional、不受欢迎的行为。后果：私下书面警告并解释违规性质，可能要求公开道歉。
2. **警告**：单次或一系列违规行为。后果：警告并附带后续限制，在指定时间内避免与相关人员互动，违反将升级处理。
3. **临时封禁**：严重违反社区标准，包括持续的不当行为。后果：在指定时间内禁止与社区进行任何互动。
4. **永久封禁**：表现出违反社区标准的固定模式，包括持续的不当行为、骚扰个人，或攻击、贬低某类群体。后果：永久禁止在社区内进行任何公开互动。
